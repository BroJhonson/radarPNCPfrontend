Política de Privacidade do App "Finnd"

Versão 1.0 – Última Atualização: 18 de setembro de 2025

1. Introdução

Esta Política de Privacidade descreve como a LaySoft One, coleta, usa e protege os dados no app "Finnd" ("App") e no site www.finnd.com.br (em desenvolvimento). Somos o controlador de dados sob a LGPD (Lei nº 13.709/2018).

O App não coleta dados pessoais no momento, focando apenas em dados públicos do PNCP. Esta Política pode ser atualizada; notificaremos via pop-up.

2. Dados Coletados

  • Dados Públicos: Apenas dados de licitações da API PNCP (ex.: editais, processos), sem dados pessoais ou sensíveis.
  • Dados Não Pessoais: Informações técnicas anônimas (ex.: dispositivo, versão do App) para melhoria.
  • Sem Coleta Pessoal: Não coletamos e-mails, localizações ou dados de pagamento. Favoritar é local; notificações requerem permissão, mas sem identificação.

No futuro, se implementarmos login ou premium, atualizaremos esta Política e obteremos consentimento.

3. Finalidades e Base Legal

Dados públicos são processados para fornecer funcionalidades (pesquisa, filtros, notificações), com base no legítimo interesse (art. 7º, IX, LGPD) e na LAI.

Armazenamento: Para performance, com retenção limitada (ativo + 1 ano pós-encerramento).

4. Compartilhamento de Dados

Não compartilhamos dados com terceiros, exceto:

  • AdMob para anúncios (dados anônimos, conforme política do Google).
  • Autoridades legais se requerido.

5. Armazenamento e Segurança

Os dados são armazenados em um servidor seguro com as medidas de proteção padrão de mercado. Dado que as informações processadas são públicas, não são aplicadas camadas de criptografia adicionais sobre os dados em repouso.

A retenção é feita enquanto a licitação está ativa mais 1 ano; depois, ocorre a exclusão automática.

6. Direitos do Usuário (LGPD)

Acesse, corrija ou exclua dados via suporte@contato.finnd.com.br. Como não coletamos dados pessoais, os direitos são limitados a dados anônimos.

7. Crianças e Privacidade

Não destinado a menores de 16 anos.

8. Transferências Internacionais

Não aplicável; dados hospedados no Brasil.

9. Contato e Encarregado

Dúvidas: suporte@contato.finnd.com.br. Encarregado (DPO): O próprio Desenvolvedor.
